Android端搭建hexo教程

在安卓上搭建hexo教程

什么是hexo?

是js运行环境，把本地md文件编译成静态html,css,js等文件
hexo官方网站

搭建:

所需环境

android+git+node.js
termux终端下载
在termux终端下执行以下命令

git配置用户名和邮箱

apt update && apt upgrade //更新apt
apt install coreutils //安装一些基本工具
apt install git //安装git
apt install nodejs //安装node.js

安装hexo

npm install hexo-cli -g //安装
mkdir hexo //创建博客文件夹 
cd hexo //进入博客文件夹 
hexo init //初始化博客

本地工作完成了，可以发布到网上，远程存储库，这里选择github

部署到github上

进入本地hexo目录下，编辑_config.yml，在末尾添加如下代码:
注: repository为要提交到的版本库的url，branch为分支

deploy: 
  type: git 
  repository: https://github.com/username/username.github.io.git 
  branch: master

git config --global user.name "John Doe" $ git config --global user.email johndoe@example.com

终端下执行下列命令

$ hexo clean //出错请输入su获取root再执行 
$ hexo generate 
$ hexo deploy //输入GitHub账户和密码即可
