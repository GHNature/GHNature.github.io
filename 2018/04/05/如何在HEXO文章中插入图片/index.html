如何在hexo文章中插入图片

方式一

配置站点根目录下_config.yml中post_asset_folder: true，
站点根目录下执行如下命令，安装hexo插件

npm install https://github.com/CodeFalling/hexo-asset-image --save

这样在执行hexo n "title"时，会在_posts目录生成一个与文章同名的文件夹，将要引用的图片放进去，在博客中便可这样引

![图片](你的文章名/timg.jpg)

"图片"为图片还未加载出来时显示的文字。

方式二

在hexo3.x中，自身加入了插件代码，方式一的基础上，便可在文章中用如下语法

{% asset_img timg.jpg 图片 %}

方式三

有时需要在不依赖图片路径的情况下也可以引用图片，用base64对图片字节码编码
base64图片在线转换
便可在文章中引用图片

![图片][img]

可在文章任何位置定义[img]:base64码，为了减轻对阅读与操作的影响可定义到文末。

也可以这样引用

![图片](base64码)

文末定义

[img]:base64码

在hexo中插入图片方式有很多，但在此主要介绍以上三种方式。
